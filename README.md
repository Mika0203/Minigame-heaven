## How to install ?

* `$ git clone https://github.com/Mika0203/Minigame-heaven`
* `$ cd Minigame-heaven`
* `$ npm install`
* `npm i pm2 -g`
* `pm2 start app.js --watch`

## Requirement

* Server : Node.js
* Database : MongoDB
