## Requirement

* Nodejs v12.18.3
* MongoDB 3.6.2

## How to install ?

* `$ git clone https://github.com/Mika0203/Minigame-heaven`
* `$ cd Minigame-heaven`
* `$ npm install`
* `$ npm i pm2 -g`
* `$ pm2 start app.js --watch`

* http://localhost:1234



## Use Docker-compose

* `$ docker-compose up`
* `$ npm install`
* `$ npm i pm2 -g`
* `$ pm2 start app.js --watch`
